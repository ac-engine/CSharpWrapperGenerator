# C#ラッパージェネレータ

ACEのEngine側ラッパーをswig生成コードから生成するツール。

## 生成ルール

まだ準拠していないルールもあります。

* 列挙体はswigと同じ構造で生成する。
* クラスは以下のルールで生成する。
    * 一部のクラスは生成しない。(ブラックリスト)
    * Coreで始まるクラスはCoreを取る。
    * ImpがつくバージョンがあるクラスはImpが付く方のみ生成する。このときImpは取る。
    * 一部のメソッドは生成しない。(ブラックリスト)
    * 一部のメソッドはpartialメソッドとして生成する。(ブラックリスト)
